国产电影大全静态站点
====================

- 影片总数：2000
- 分类页：10 个
- 详情页：2000 个
- 播放源：20 个 m3u8 地址（来自原始素材）

目录结构：
- index.html：首页
- categories.html：分类总览
- ranking.html：热播榜
- category-01.html ~ category-10.html：独立分类页
- movie-0001.html ~ movie-2000.html：影片详情页
- assets/app.css、assets/app.js：共享样式与交互
- assets/source/：原始素材备份
